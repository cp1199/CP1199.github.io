# 青岛旅游攻略｜红瓦绿树，碧海蓝天3日逛吃指南
## 前言
青岛兼具欧式老城风情与治愈海岸线，节奏舒缓、拍照出片，3天行程覆盖人文、海景、啤酒文化、地道海鲜，新手无踩坑路线。

## 一、出行基础指南
### 1. 最佳出行时间
- 6-8月：海水适宜下海玩水，海边日落绝美（旺季人多）
- 9-10月：气温舒适，游客少，八大关秋景封神
- 11-次年3月：栈桥可喂海鸥，适合低成本拍照

### 2. 住宿推荐（优先选市南区）
1. **中山路/栈桥附近**：交通便利，老城景点步行可达，平价民宿多
2. **八大关/太平角**：海景酒店，安静轻奢，预算偏高
3. **五四广场/奥帆中心**：现代商圈，夜景好看，地铁便捷
❌ 避雷：台东步行街住宿嘈杂，不推荐长期住

### 3. 交通贴士
- 地铁覆盖绝大多数景点，支付宝领取青岛地铁码
- 老城小路多，短途骑行共享单车最舒服
- 海边景点串联：栈桥→小鱼山→八大关→奥帆，沿海公交直达

## 二、3天经典行程（不赶路版）
### Day1 老城漫游｜红瓦欧式老街
**路线：栈桥 → 圣弥厄尔大教堂 → 银鱼巷 → 小鱼山公园 → 大学路 → 琴屿路日落**
1. **栈桥（免费）**
青岛地标，清晨来人少，退潮可赶海挖小蛤蜊；秋冬能喂海鸥，不建议在岸边大排档吃海鲜，溢价高。
2. **圣弥厄尔大教堂（外观免费，内部10元）**
哥特式双塔建筑，红墙白鸽超好拍，门外打卡足够，不用特意买票进内部。
3. **银鱼巷/广兴里**
老里院改造文艺街区，复古小店、咖啡馆，适合买青岛冰箱贴伴手礼。
4. **小鱼山公园（门票10元）**
青岛性价比最高观景台，360°俯瞰红瓦屋顶+整片海湾，人比信号山更少。
5. **大学路网红墙**
红蓝转角打卡点，清晨/傍晚光线柔和，避开正午暴晒；隔壁海大鱼山校区爬山虎墙超出片。
6. **琴屿路+小青岛**
S型滨海公路，蓝调时刻灯塔日落氛围感拉满，宫崎骏同款海边小路。

**晚餐：中山路九龙餐厅（本地人私藏海鲜小馆）**
必点：辣炒蛤蜊、海鲜疙瘩汤、蚝油扇贝，人均50元，平价地道。

---
### Day2 海岸风光+啤酒文化
**路线：八大关 → 第二海水浴场 → 青岛啤酒博物馆 → 台东夜市**
1. **八大关（免费）**
万国建筑博览群，200多栋各国别墅，推荐打卡花石楼、公主楼；秋天梧桐红叶绝美，可租双人单车骑行。
2. **第二海水浴场**
沙滩细腻，海边堤坝拍照绝佳，人少清净，适合踩水散步。
3. **青岛啤酒博物馆（门票60元）**
了解百年青啤酿造工艺，门票附赠一杯原浆啤酒+一杯纯生，还有醉酒小屋体验，16:30停止入场记得早去。
4. **台东步行街夜市**
青岛烟火气天花板，小吃密集，分多次少量购买避免吃撑。

**夜市必吃：烤鱿鱼、脂渣、海鲜馄饨、西镇臭豆腐**

---
### Day3 海岛治愈/崂山二选一
#### 方案A（轻松看海，懒人首选）
小麦岛公园 → 燕儿岛山公园 → 奥帆中心情人坝 → 五四广场灯光秀
- 小麦岛：草坪+海岸线，野餐、看日落绝佳，城市天际线同框
- 燕儿岛：礁石洞穴、木栈道，小众网红拍照地
- 奥帆情人坝：远眺浮山湾，傍晚看海上日落
- 五四广场「五月的风」，每晚19:30/20:00两场灯光秀，对岸情人坝观赏视角最佳

#### 方案B（山海全景，体力充足）
崂山仰口景区一日游（门票90元）
海上第一名山，登山看海，青山渔村彩色房屋错落，山海相连视野震撼；建议早出发，预留一整天。

## 三、青岛必吃美食清单
### 海鲜硬菜
1. 辣炒蛤蜊：青岛餐桌标配，鲜辣入味
2. 鲅鱼水饺：双合园老字号，皮薄馅大，海鲜水饺天花板
3. 海肠捞饭：鲜掉眉毛，鲁菜特色
4. 清蒸扇贝/生蚝：原汁原味，海边必点

### 本地特色小吃
1. **袋装鲜啤**：本地人标配，塑料袋装原浆啤酒，路边配烧烤
2. 脂渣：五花肉熬制酥脆肉干，下酒零食
3. 流亭猪蹄：卤至软糯，可真空打包当伴手礼
4. 青岛锅贴：墨鱼/虾仁爆汁锅贴，劈柴院老店正宗
5. 西镇臭豆腐：外酥里嫩，秘制酱料不臭超香

### 本地人推荐老店
- 海鲜家常菜：九龙餐厅、双合园饺子
- 夜宵小吃：海平馄饨、如意香辣鸡架
- 伴手礼：鑫复盛流亭猪蹄（真空礼盒）

## 四、避坑指南
1. 栈桥、劈柴院景区海鲜大排档慎吃，价格虚高，容易缺斤短两
2. 海边拍照不要轻信路边收费摄影师，自带相机即可
3. 崂山不要报低价一日游团，强制消费，自行地铁/打车前往更稳妥
4. 夏季下海注意潮汐，礁石区域小心打滑划伤
5. 网红打卡点不用扎堆排队，周边小路拍照效果更好

## 五、伴手礼推荐
流亭猪蹄、袋装青岛原浆啤酒、崂山绿茶、海鲜干货、复古青岛文创冰箱贴
